# SOMA v3 Agent Prompt Pack

이 폴더의 각 파일은 에이전트에게 그대로 전달 가능한 독립 프롬프트다.
각 파일은 공통 규칙 + 담당 이슈 지시를 포함한다.

## 전달 순서
1. `01-agent-lead-wave0-soma-vbj.4.md`
2. `02-agent-core-wave1-soma-vbj.5.md`
3. `03-agent-provider-wave1-soma-vbj.6.md`
4. `04-agent-telegram-wave1-soma-vbj.7.md`
5. `05-agent-slack-wave1-soma-vbj.8.md`
6. `06-agent-storage-wave1-soma-vbj.10.md`
7. `07-agent-lead-wave2-soma-vbj.9.md`
8. `08-agent-qa-wave3-soma-vbj.11.md`
9. `09-agent-lead-wave4-soma-vbj.12.md`
10. `10-agent-optional-track-soma-vbj.13.md`

## 사용법
- 각 파일의 `Copy/Paste Prompt` 블록 전체를 담당 에이전트에게 전달한다.
- 비리팩토링 이슈는 `soma-wmy`에 의해 차단되어 있으므로 건드리지 않는다.
